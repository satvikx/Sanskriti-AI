# Sanskriti AI WhatsApp Chatbot
Sanskriti AI is a Multilingual Chatbot-based Ticketing System, integrated with WhatsApp, will address these issues by streamlining the ticketing process, improving accuracy, and offering support in multiple languages.

### Tech Used
Backend Server - Django <br />
API - Meta Business API (WhatsApp Integration), Gemini (NLP), PhonePe (Payments)<br />
Webhooks - ngrok
